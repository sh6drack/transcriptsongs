# transcriptsongs

shazam but for dj sets. upload your mix and get timestamps.

## setup

```bash
./setup.sh
```

get api key from https://audd.io/ (free)

add it to `.env`

## run

```bash
source venv/bin/activate
python3 app.py
```

open http://localhost:5001
